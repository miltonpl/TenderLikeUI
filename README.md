## About
- The simple app is about swiping the card view to left to remove and to the right to save
-  Move over every card view has imformation details about the user, like profile , contact, location
- On top there are a bar botton on left to dislike the current  user, and on the right to save the user to the favorite list
- On the botton there there two bottons one to display the favorite list and the other to show the card screen stock.
